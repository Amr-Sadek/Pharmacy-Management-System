# 💊 Pharmacy Management System

### Software Engineering Analysis & Design Project

</div>

---

## 📖 Overview

This project was developed as part of the Software Engineering course at Future University in Egypt.

The project focuses on the analysis and design of an Online Pharmacy Management System, aiming to improve inventory management, customer communication, online medicine ordering, and pharmacy administration.

---

## ✨ Proposed Features

- Online Pharmacy Store
- Inventory Management
- Prescription Upload
- Barcode Scanning
- Customer Messaging
- Online Ordering
- Multiple Payment Methods
- Sales Reports & Analytics
